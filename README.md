# Joanna AI Coursework

AI/ML coursework repository for Joanna, conducted by Himanshu Singh under ModernAgeCoders.

This repository contains lecture notes, coding exercises, implementations, assignments, and learning resources covered throughout the course.

## Course Overview

This repository documents the complete learning journey across core Artificial Intelligence and Machine Learning topics including:

- Gradient Descent
- Machine Learning Fundamentals
- Deep Neural Networks
- Convolutional Neural Networks (CNNs)
- Reinforcement Learning
- Autonomous Driving Basics
- PID Control Systems
- GPT Architecture
- Generative AI Concepts
- Practical Coding Exercises
- Simulation & Racing Practice
